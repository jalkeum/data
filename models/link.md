# Link

`color` 키에 사용할 수 있는 값은 아래 값들로 한정.

- `"midnight-blue"`
- `"indigo"`
- `"british-racing-green"`
- `"bright-red"`
- `"tangerine2"`
- `"blue-lagoon"`
- `"raw-umber"`
- `"sapphire"`

| 키 | 값 | 필수 | 비고 |
| --- | --- | --- | --- |
| text | `string` | true | |
| url | `string` | true | |
| color | `string` | true | 한정된 값만 사용 가능 |
